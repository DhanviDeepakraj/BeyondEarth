# BeyondEarth
Beyond Earth – Space Exploration and Astronomy Website

1. Project Overview
Beyond Earth is an educational astronomy website that provides information
about space exploration, celestial events, constellations,
astrophotography, and recent developments in space science.

2. Live Demo
Use:
https://beyond-earth-nine.vercel.app/

3. Features
• Responsive Navigation Bar
• Celestial Events Section
• Constellations Section
• Astrophotography Section
• Space News Section
• Interactive Clickable Cards
• Responsive Design
