# Beyond Earth 🚀

## Space Exploration and Astronomy Website

Beyond Earth is an educational astronomy website that provides information about space exploration, celestial events, constellations, astrophotography, and recent developments in space science.

## 🌐 Live Demo

https://beyond-earth-nine.vercel.app/

## ✨ Features

- Responsive Navigation Bar
- Celestial Events Section
- Constellations Section
- Astrophotography Section
- Space News Section
- Interactive Clickable Cards
- Responsive Design

## 🛠️ Technology Stack

- HTML5
- CSS3
- JavaScript
- GitHub Pages
- Vercel
- Visual Studio Code

## 📁 Project Structure

```text
index.html                → Home Page
box1.html                 → Celestial Events
box2.html                 → Constellations
box3.html                 → Astrophotography
right-image-page.html     → Space News
style.css                 → Styling
script.js                 → Navigation Logic
```

## 🚀 How to Run

1. Clone the repository

```bash
git clone <repository-url>
```

2. Open the project folder

3. Launch `index.html` in your browser
